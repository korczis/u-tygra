Správná teplota servírování je jedním z nejdůležitějších faktorů pro chuťový zážitek z piva. **Příliš studené** pivo ztrácí aróma a chuť. **Příliš teplé** je nepříjemné a nadměrně se pění.

## Doporučené teploty

| Styl | Teplota | Proč |
|------|---------|------|
| Light Lager | 3–5 °C | Osvěžení, čistota |
| Pilsner / Český ležák | 6–8 °C | Chmelové aróma, svěžest |
| Pšeničné / Witbier | 6–8 °C | Kvasnicové aróma |
| Pale Ale / APA | 8–10 °C | Chmelové tóny |
| IPA / NEIPA | 8–12 °C | Tropické aróma |
| Amber / Polotmavé | 8–10 °C | Sladové tóny |
| Tmavý ležák | 8–10 °C | Karamelová komplexnost |
| Porter | 10–12 °C | Pražené tóny |
| Stout | 10–14 °C | Čokoládová plnost |
| Belgian Ale | 10–14 °C | Esterové aróma |
| Barley Wine | 12–16 °C | Komplexní, zahřívající |
| Imperial Stout (BA) | 14–16 °C | Dřevěné tóny |
| Sour / Lambik | 6–10 °C | Svěží kyselost |

## Základní pravidlo

**Čím silnější a tmavší pivo, tím vyšší teplota.** Lehké ležáky studenější, silné tmavé teplejší.

## Proč teplota tak záleží

- **Nízká teplota** (pod 5 °C): Tlumí chuťové receptory, pivo chutná „vodově"
- **Správná teplota**: Uvolňují se aromatické sloučeniny, vyvážená chuť
- **Vysoká teplota** (nad 16 °C): Alkohol je příliš výrazný, pivo se nadměrně pění
