**ABV** udává procentuální obsah alkoholu v pivu.

| Styl | ABV |
|------|-----|
| Výčepní (desítka) | 3,5–4,5 % |
| Světlý ležák | 4,0–5,5 % |
| IPA | 5,5–7,5 % |
| Stout | 4,0–12,0 % |
| Barley Wine | 8,0–14,0 % |
