**ABV** (Alcohol by Volume) je mezinárodně uznávaný způsob vyjádření obsahu alkoholu v nápoji. Udává se v procentech objemu — kolik procent celkového objemu piva tvoří čistý etanol.

## Jak ABV vzniká

Alkohol v pivu vzniká při **[kvašení](@/glosar/kvaseni.md)**. Kvasnice konzumují cukry z [mladiny](@/glosar/mladina.md) a přeměňují je na etanol a CO₂. Čím více cukru v mladině (vyšší stupňovitost), tím vyšší může být výsledné ABV.

## Typické hodnoty ABV

| Pivní styl | ABV | Charakter |
|------------|-----|-----------|
| Výčepní (desítka) | 3,5–4,5 % | Lehké, pitné, na každý den |
| Světlý ležák (12°) | 4,0–5,5 % | Klasický český standard |
| Tmavý ležák | 3,5–5,5 % | Karamelová sladkost |
| Pale Ale / APA | 4,5–6,0 % | Chmelové aróma |
| IPA | 5,5–7,5 % | Intenzivní hořkost |
| Pšeničné | 4,5–5,5 % | Lehké, banánové tóny |
| Stout / Porter | 4,0–7,0 % | Pražené slady, čokoláda |
| Imperial Stout | 8,0–12,0 % | Velmi silné, komplexní |
| Barley Wine | 8,0–14,0 % | Nejsilnější pivní styl |

## Jak se ABV měří

1. Změří se hustota mladiny před kvašením (OG — Original Gravity)
2. Změří se hustota po kvašení (FG — Final Gravity)
3. **ABV = (OG - FG) × 131,25**

## ABV vs. stupňovitost

V Česku se tradičně používá **[stupňovitost](@/glosar/stupnovitost.md)** (°Plato), která měří obsah extraktu PŘED kvašením. ABV měří alkohol POTÉ.

- 10° ≈ 3,5–4,5 % ABV
- 12° ≈ 4,5–5,5 % ABV
- 14° ≈ 5,5–6,5 % ABV

## Vliv na chuť

- **Nízké (do 4,5 %)**: Svěží, pitné. Letní osvěžení.
- **Střední (4,5–6,0 %)**: Vyvážené, plnější tělo.
- **Vysoké (6,0–9,0 %)**: Intenzivní, zahřívající.
- **Velmi vysoké (9,0 %+)**: Komplexní, dezertní charakter.

## Zajímavosti

- Nejsilnější pivo světa (Snake Venom) má údajně 67,5 % ABV
- Bezalkoholové pivo má do 0,5 % ABV
- Česká legislativa definuje pivo jako nápoj s min. 0,5 % ABV
