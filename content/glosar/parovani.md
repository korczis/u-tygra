**Párování piva s jídlem** (food pairing) je umění nacházení ideálních kombinací chutí mezi pivem a pokrmem.

## Tři principy párování

### 1. Doplňování (Complement)
Podobné chutě se vzájemně posilují:
- Karamelový stout + čokoládový dort
- Pšeničné pivo + lehký citrusový salát

### 2. Kontrast (Contrast)
Opačné chutě se vyrovnávají:
- Hořká IPA + sladké BBQ maso
- Kyselý Sour + tučný sýr

### 3. Vyčištění (Cleanse)
Pivo „čistí" patro mezi sousty:
- Perlivý Pilsner + smažený sýr (odřezává mastnotu)
- Kyselá Gose + tučné maso

## Průvodce párováním

| Pivní styl | Ideální jídlo | Proč to funguje |
|------------|---------------|-----------------|
| **Světlý ležák** | Svíčková, řízek, smažený sýr | Čistí mastnotu, osvěžuje |
| **Tmavý ležák** | Pečená kachna, guláš | Karamel doplňuje pečení |
| **IPA** | Burger, curry, pikantní jídla | Hořkost kontrastuje pálivost |
| **NEIPA** | Sushi, ceviche, lehké saláty | Ovocnost doplňuje svěžest |
| **Pšeničné** | Ryby, saláty, bílý sýr | Lehkost doplňuje lehkost |
| **Stout** | Čokoládový dort, ústřice | Pražené tóny + slanost/sladkost |
| **Sour / Gose** | Chevrè, mořské plody | Kyselost řeže tučnost |
| **Belgické** | Měkké sýry, mušle | Kořeněné tóny doplňují |
| **Porter** | Grilované maso, BBQ | Kouřovost + kouřovost |
| **Barley Wine** | Silné sýry, foie gras | Síla odpovídá síle |

## Česká pivní kuchyně

Tradiční česká jídla jsou přirozeně spárovaná s českým pivem:
- **Utopenec** + Výčepní desítka
- **Tlačenka s cibulí** + Světlý ležák
- **Nakládaný hermelín** + Polotmavé
- **Svíčková na smetaně** + Dvanáctka
- **Vepřo knedlo zelo** + Plzeňský ležák
- **Smažený sýr** + Čerstvě načepovaný ležák

## Zajímavosti

- V Belgii existují pivní restaurace (bistronomie) na úrovni michelinských
- Japonské pivo Asahi má speciálně navrženou řadu pro sushi pairing
- Čokoládový stout + vanilková zmrzlina = pivní dezert roku
