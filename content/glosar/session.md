**Session** pivo má nižší obsah alkoholu (typicky do 4,5 % ABV), ale zachovává plnou chuť svého „silnějšího bratříčka".

## Původ názvu

Termín pochází z **britské pracovní tradice** — dělníci měli povolené pivní přestávky (*sessions*) během pracovního dne. Potřebovali piva, která je neopijí, ale přitom budou chutnat.

Jiná teorie: za obou světových válek britská vláda omezila obsah alkoholu v pivu — vznikly „session" verze.

## Typické session styly

| Styl | ABV | Charakter |
|------|-----|-----------|
| Session IPA | 3,5–4,5 % | Chmelové aróma, nízký alkohol |
| Session Pale Ale | 3,5–4,5 % | Vyvážená, pitná |
| Mild | 3,0–3,8 % | Anglický, sladový |
| Berliner Weisse | 2,8–3,5 % | Kyselá, lehká |
| Gose | 3,5–4,5 % | Slaná, kyselá |
| Český výčepní (10°) | 3,5–4,5 % | Klasická česká desítka |

## Session IPA

Nejpopulárnější session styl. Výzva: jak zachovat intenzivní [chmel](@/glosar/chmel.md)ové aróma při nízké [slad](@/glosar/slad.md)ové páteři? Řešení: masivní [dry hopping](@/glosar/dry-hopping.md) a speciální techniky (BIAB, turbid mashing).

## Zajímavosti

Česká **desítka** (10° výčepní pivo) je vlastně tradiční session pivo — lehké, pitné, na každý den. Češi pili „session" pivo staletí, než se termín stal módní!
