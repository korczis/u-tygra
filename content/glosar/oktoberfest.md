**Oktoberfest** se koná na mnichovské louce Theresienwiese od poloviny září do prvního říjnového víkendu.

## Fakta

| | |
|---|---|
| Založen | 1810 (svatba prince Ludvíka) |
| Délka | 16–18 dní |
| Návštěvníci | ~6 milionů ročně |
| Spotřeba piva | ~7,7 milionů litrů |
| Pivovary | Pouze 6 mnichovských (Augustiner, Paulaner, Spaten, Hacker-Pschorr, Hofbräu, Löwenbräu) |

## Oktoberfest-pivo

Tradičně jantarový **Märzen**, ale od 90. let je většina piv světlejší, blíže Helles — nazývá se *Festbier* (5,8–6,3 % ABV).
